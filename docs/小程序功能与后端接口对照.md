```

```

## 七、常见问题

**Q：点击 AI 问诊或缴费提示请先登录？**  
A：需先完成注册/登录；Tab 页不会自动跳转，点按钮时会引导登录。

**Q：一键缴费失败？**  
A：确认后端 `POST /payment/create` 接受的字段（当前传 `patientId` + `itemIds`），与后端 DTO 对齐。

**Q：签到按钮不可用？**  
A：仅挂号状态为 `PAID` / `PENDING` / `WAITING` 时显示，且需后端允许当日签到。
